2.4. docker system

docker system 是 Docker 提供的一个顶级管理命令，用来查看和清理 Docker 的整体资源（镜像、容器、卷、构建缓存等）。

2.4.1. 查看 Docker 整体资源占用

类似 df -h，会显示：

  • 镜像大小
  • 容器占用空间
  • 卷占用空间
  • 构建缓存大小

2.4.2. 清理未使用资源

会删除：

  • 已停止的容器
  • 未使用的网络
  • 悬空的镜像（没有 tag 的 <none> 镜像）
  • 构建缓存
